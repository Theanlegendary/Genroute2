02 · Choose the input
Fastest route to an accurate result

Use the fastest reliable input—not the most familiar one.

Staff often type the market name because it feels easy. A map link or branch code is usually faster because it removes ambiguity.

Recommended order

Map link → Branch code → Coordinates → Place name + province

Widen the search carefully
